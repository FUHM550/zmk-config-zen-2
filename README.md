---

- `LWR` → Lower layer  
- `RSE` → Raise layer  
- `GUI` → Cmd / Win  
- `PLY` → Play/Pause  
- `BT_*` → Bluetooth controls  
- `M_direction` → Pointer controls
- `S_direction` → Scroll controls
